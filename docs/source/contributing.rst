Contribute to FP-Tools
=======================

Thank you for your interest in FP-Tools! We welcome contributions of all kinds, 
including bug reports and feature requests, code improvements and new analysis 
modules, documentation improvements, and testing/validation.

Getting Started
---------------

1. **Fork the repository** on GitHub and clone your fork locally.
2. **Create a new branch** for your work.
3. **Make your changes** and commit them with clear messages, then push them to 
   your fork.
4. Open a pull request between <new branch> and main
