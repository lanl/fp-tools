Contact Us
==========

We'd love to hear from you! Whether you have questions, suggestions, bug 
reports, or want to contribute, please reach out to us through any of the 
following channels:

Issue Tracker
-------------

If you encounter a bug, have a feature suggestion, or want to ask a question, 
please open an issue on our GitHub repository:
`Github Issue Tracker <https://github.com/lanl/fp-tools/issues>`_.

Please use the issue tracker for:

- Bug reports (e.g. crashes, incorrect results)
- Feature suggestions
- Questions about intended usage or behavior

If you're submitting a bug report, include the following when possible:

- Steps to reproduce the problem
- Your configuration file
- Any error messages or logs

If you're submitting a feature request, be clear about:

- What you'd like the tool to do
- Why it would be usefule
- Any relevant examples or prior work

If you're not sure whether your issue fits, go ahead and open one and we'll 
help you sort it out. :)
