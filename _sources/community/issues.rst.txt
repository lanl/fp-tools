Issue Tracker
=============

If you encounter a bug, have a feature suggestion, or want to ask a question,
please open an issue on out GitHub repository: 
`Github Issue Tracker <https://github.com/lanl/fp-tools/issues>`_.

When to Open an Issue
---------------------

Please use the issue tracker for:

- Bug reports (e.g. crashes, incorrect results)
- Feature suggestions
- Questions about the intended usage or behavior

If you're not sure whether your issue fits, go ahead and open one and we'll 
help you sort it out.

How to Report a Bug
-------------------

Include the following in your bug report when possible:

- Steps to reproduce the problem
- Your configuration file
- Any error messages or logs

Tips for Feature Requests
-------------------------

Be clear about:

- What you'd like the tool to do
- Why it would be useful
- Any relevant examples or prior work
